# Mohu App

一个simple的膜乎免 番羽土啬 Windows & Linux APP

[![+1s](https://img.shields.io/badge/%CE%98..%CE%98-%2B1s-green.svg)](https://en.wikipedia.org/wiki/Moha_culture)

## 功能

* 自动添加hosts, 免 番羽土啬 浏览膜乎

* 内置三个镆铪/乳苞小游戏:

    * Flappy Winnie
    * Flappy Frog
    * 切包子

* 自动检查更新
